== Description ==

스마틀리는 보다 똑똑한 제품소개를 위해 개발 되었습니다.
제품을 직접 판매하지 않는 홈페이지 제작에 유용합니다.

데모 페이지 : http://demo-homepageep.wpcorona.com/smartly

= 주요 기능 =

* 일반 페이지가 아닌, ‘제품’이라는 문서(포스트)를 만들어 줍니다.
* ‘제품’ 대신 ‘서비스’나 ‘상품’ 등으로 이름을 변경해 사용할 수 있습니다.
* 카테고리와 태그로 제품을 분류할 수 있습니다.
* 별도의 제품소개서(브로셔)가 있다면, 파일로 첨부할 수 있습니다.

= 스마틀리 활용 =

* 스마틀리는 템플릿을 포함하지 않습니다.
* Divi Builder나 템플릿 개발을 통해 자유롭게 디자인을 만드시기 바랍니다.
* 스마틀리는 워드프레스 개발원칙을 준수합니다.
* ACF를  이용해, 자유롭게 제품 DB를 구성할 수 있습니다.
* SEO 플러그인을 이용해, 제품 소개에 한해 별도의 SEO 설정을 적용할 수 있습니다.

== Installation ==

1. 플러그인을 활성화 합니다
2. 메뉴의 제품 > 설정 에서 슬러그를 변경할수 있습니다
